# image-analysis

Image and video segmentation tools for Claude Code.

## Skills

### sam3-api

Provides API patterns for SAM3 (Segment Anything Model 3) from Facebook Research:

- Text prompts for open-vocabulary segmentation
- Bounding box prompts (positive/negative)
- Point prompts (SAM1-style interactive)
- Video tracking across frames
- Batch inference API

**Trigger**: Use when working with SAM3 for image/video segmentation tasks.

## Installation

```bash
/plugin install image-analysis@claude-plugins
```
